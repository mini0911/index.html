// 🟢 智慧連動雲端 + 🚀 企業級高容量綠色通道（彻底解決連線超時、封鎖問題） function fetchCloudAttendance() { const debugEl = document.getElementById('debug-banner'); const diagEl = document.getElementById('diagnostic-log'); if (!customConfig.csvUrl) return; let targetUrl = customConfig.csvUrl; // 自動修正網址格式 if(targetUrl.includes('/pub?')) { targetUrl = targetUrl.replace('/pub?', '/export?format=csv&'); } else if (targetUrl.includes('/edit')) { // 防呆機制：如果你不小心複製到一般的試算表網址，系統會自動幫你硬轉成 CSV 格式網址！ targetUrl = targetUrl.split('/edit')[0] + '/export?format=csv'; } // 🌟 核心升級：改用 cors-anywhere 或微軟推薦的全新穩定的公用跳板，免疫超時與被擋 const backupFetchUrl = "https://corsproxy.io/?" + encodeURIComponent(targetUrl); fetch(backupFetchUrl) .then(res => { if (!res.ok) throw new Error(`雲端伺服器回應錯誤 (狀態碼: ${res.status})，請確認 Google 後台是否有勾選發布為 CSV`); return res.text(); // 直接解出文字，不再經由 contents 轉手 }) .then(csvText => { if(!csvText || csvText.trim().length < 10) { if (diagEl) diagEl.innerText = "⚠️ 警告：從雲端下載的試算表內容為空，請確認表單目前是否有人填寫。"; return; } const lines = csvText.split(/\r?\n/); let dataChanged = false; let matchCount = 0; // 面板簡化日期陣列 (例如: ["7/16", "7/17", "7/18"...]) const formattedPanelDates = customConfig.dates.map(d => { const parts = d.replace(/-/g, '/').split('/'); if (parts.length >= 3) { return `${parseInt(parts[1], 10)}/${parseInt(parts[2], 10)}`; } return d.trim(); }); // 🛠️ 診斷回報面板資訊初始化 let debugInfo = `【🔍 雲端數據大腦診斷中】\n`; debugInfo += `網頁期望比對的簡化日期：[${formattedPanelDates.join(', ')}]\n`; debugInfo += `試算表總列數：${lines.length} 列\n\n`; debugInfo += `▼ 雲端前 3 筆填表資料的精準分析如下：\n`; let trackedLinesCount = 0; for (let i = 1; i < lines.length; i++) { let line = lines[i].trim(); if (!line) continue; const cells = line.split(/,(?=(?:(?:[^"]*"){2})*[^"]*$)/); if (!cells || cells.length < 1) continue; // 抓取第 A 欄：時間戳記 let rawTimestamp = cells[0].replace(/"/g, '').replace(/\s/g, ''); let extractedRowDate = ""; let match = rawTimestamp.match(/(\d{2,4})[\/\-](\d{1,2})[\/\-](\d{1,2})/); if (match) { extractedRowDate = `${parseInt(match[2], 10)}/${parseInt(match[3], 10)}`; } else { customConfig.dates.forEach((d, idx) => { if (rawTimestamp.includes(formattedPanelDates[idx])) extractedRowDate = formattedPanelDates[idx]; }); } let targetPanelDateIndex = formattedPanelDates.indexOf(extractedRowDate); // 抓取 C 欄學號 let cloudStudentId = cells[2] ? cells[2].replace(/"/g, '').trim() : "（無資料）"; // 收集診斷前 3 筆 if(trackedLinesCount < 3) { trackedLinesCount++; debugInfo += `[第 ${i} 列紀錄] ➔ A欄時間戳記: "${cells[0]}", 系統判定日期為: "${extractedRowDate || '無法辨識'}"\n`; debugInfo += ` ➔ B欄電子郵件: "${cells[1] || ''}"\n`; debugInfo += ` ➔ C欄(第3欄)學號: "${cloudStudentId}" ➔ 比對結果: `; if (targetPanelDateIndex === -1) { debugInfo += `❌ 日期不符合目前設定的 7 個日期！被系統過濾掉。\n\n`; } else { const testStudent = currentStudents.find(student => student.id === cloudStudentId); if (!testStudent) { debugInfo += `❌ 日期符合(${customConfig.dates[targetPanelDateIndex]})，但此學號不在下方的 308 人名單中！\n\n`; } else { debugInfo += `✅ 日期符合、學號正確！應該要打勾。\n\n`; } } } // 2. 真正進行比對打勾邏輯 if (targetPanelDateIndex !== -1 && cells[2]) { let matchedActualDate = customConfig.dates[targetPanelDateIndex]; const foundStudent = currentStudents.find(student => student.id === cloudStudentId); if (foundStudent) { if(foundStudent.attendance[matchedActualDate] !== 1) { foundStudent.attendance[matchedActualDate] = 1; dataChanged = true; } matchCount++; } } } if (diagEl) diagEl.innerText = debugInfo; if(matchCount > 0) { debugEl.innerHTML = `🟢 [雲端同步成功] 系統已與 Google C 欄對齊！目前共自動追蹤勾選了 ${matchCount} 人次！`; debugEl.style.borderLeftColor = "#00e676"; } else { debugEl.innerHTML = `🟢 [雲端連線成功] 通道暢通！目前 C 欄尚未成功勾選任何人（請看下方黃色面板診斷原因）。`; debugEl.style.borderLeftColor = "#ffeb3b"; } if (dataChanged) { renderTable(); calculateStats(); } }) .catch(e => { console.error(e); debugEl.innerHTML = "❌ [異常提示] 雲端通道連線超時，請點開設定面板確認網址是否為「發布為 CSV」的網址。"; if (diagEl) diagEl.innerText = "❌ 診斷失敗：" + e.message; }); }
